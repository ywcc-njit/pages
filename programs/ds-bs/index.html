BS in Data Science Curriculum

NJIT's Bachelor of Science in Data Science program equips students with the rigorous mathematical foundation, computational skills, and domain expertise required to excel in today's data-driven world. This carefully structured curriculum progresses from foundational mathematics and computing principles through advanced machine learning techniques and big data applications, with specialized concentration tracks in artificial intelligence, data mining algorithms, biology, finance, and mathematics. Students develop practical expertise through hands-on laboratory sessions, working with real-world datasets and implementing end-to-end data science solutions. Designed in alignment with industry needs and academic best practices, this program prepares graduates to solve complex problems across diverse sectors—from healthcare and financial services to technology and scientific research—with both technical proficiency and ethical responsibility. Whether pursuing careers as data scientists, analysts, or researchers, or continuing to advanced studies, NJIT students gain the comprehensive skill set needed to derive meaningful insights from data and make significant contributions to their fields.

New Course Descriptions

DS 113: Computing Foundations for Data Science

Prerequisites: DS 100 or equivalent programming experience
Corequisite: MATH 112 recommended

Introduction to computational methods for data science using Python. Topics include implementing numerical integration techniques, solving application problems using definite integrals, computing infinite series approximations, working with exponential and logarithmic functions in data contexts, analyzing transcendental functions, visualizing polar coordinates, and computational approaches to conic sections. Laboratory sessions provide hands-on experience applying Python libraries such as NumPy, SciPy, Matplotlib, and SymPy to reinforce calculus concepts through programming.

DS 114: Computing Foundations for Data Science II

Prerequisites: DS 113 and MATH 112 with a grade of C or better
Corequisite: MATH 337

Computational techniques in Python for data science with emphasis on linear algebra implementations and applications. Topics include matrix operations and manipulation using NumPy, implementing vector space operations, coding linear transformations, solving systems of linear equations programmatically, computing eigenvalues and eigenvectors, and applying these concepts to data science problems. Laboratory sessions reinforce mathematical concepts by visualizing abstract linear algebra operations, demonstrating the geometric interpretation of transformations, providing concrete computational examples of theoretical concepts, and showing how matrix decompositions connect to real-world data analysis.

DS 240: Data Analytics with Python

Prerequisites: None

Comprehensive introduction to data analytics using Python. Topics include advanced data manipulation and preprocessing techniques using pandas, data cleaning and transformation, exploratory data analysis, visualization with Matplotlib and Seaborn, and applying statistical methods for pattern recognition. Students will gain practical experience implementing black-box analytics tools including regression models, classification algorithms, clustering methods, and time series analysis. Laboratory sessions provide hands-on experience with real-world datasets, focusing on the entire analytics workflow from data acquisition to visualization and interpretation. Emphasis on best practices for reproducible analysis, assessing model performance, and communicating insights effectively through data storytelling. Course projects require students to develop end-to-end analytics solutions for business and scientific applications.

DS 288: Programming with APIs

Prerequisites: DS 113 or equivalent Python programming experience

Introduction to Application Programming Interfaces (APIs) with emphasis on data retrieval and analysis in Python. Topics include RESTful API fundamentals, authentication methods, request-response cycles, data parsing, and error handling. Students will gain practical experience interacting with various API services including Large Language Models (OpenAI, Anthropic), financial data sources (Alpha Vantage, Yahoo Finance), social media platforms, and public data repositories. The course introduces advanced programming concepts in an LLM-aided environment, with supplementary modules in Java and C for cross-language API implementation. Laboratory sessions focus on building real-world applications that leverage APIs for data collection, natural language processing, automated trading strategies, and intelligent agents. Students will complete projects demonstrating their ability to design, implement, and document API-driven solutions for contemporary data challenges.

DS 331: Elements of Data Management

Prerequisites: DS 240 or equivalent data analytics experience

Comprehensive introduction to modern data management principles and practices with focus on business intelligence applications. Topics include data warehouse design and implementation, dimensional modeling, ETL (Extract, Transform, Load) processes, OLAP operations, and cloud-based data storage solutions. Students will gain practical experience with spreadsheet tools for data manipulation, analysis, and visualization, including advanced formulas, pivot tables, and dashboard creation. The course introduces application development using JavaScript and spreadsheet-integrated scripting to create interactive data solutions and custom functions that extend spreadsheet capabilities. Laboratory sessions provide hands-on experience with cloud platform tools for creating data pipelines, implementing data governance practices, and developing browser-based data applications. Students will complete projects demonstrating their ability to design and implement end-to-end data management solutions that connect cloud data sources to actionable business intelligence.

DS 335: Unsupervised Algorithms for Data Science

Prerequisites: DS 240

Comprehensive exploration of unsupervised learning algorithms and their applications in data science. Topics include clustering techniques (K-means, hierarchical, DBSCAN, Gaussian mixture models), dimensionality reduction methods (PCA, t-SNE, UMAP), anomaly detection, association rule learning, and latent variable models. The course examines theoretical foundations including distance metrics, density estimation, manifold learning, and information theory while emphasizing practical implementation in Python. Students will learn to evaluate unsupervised model performance, interpret results, and select appropriate algorithms for different data types and problem domains. Laboratory sessions provide hands-on experience applying these techniques to real-world datasets in areas such as customer segmentation, image processing, recommendation systems, and text analysis. Course projects require students to design and implement end-to-end unsupervised learning pipelines, from data preparation through analysis and visualization to insight generation.

DS 375: Supervised Machine Learning with Applications

Prerequisites: DS 240

Comprehensive study of supervised learning algorithms and methodologies for predictive modeling. Topics include linear and logistic regression, decision trees, random forests, gradient boosting, support vector machines, neural networks, and ensemble methods. The course explores key concepts including bias-variance tradeoff, overfitting, regularization techniques, cross-validation, and hyperparameter tuning. Students will gain practical experience implementing these algorithms in Python using scikit-learn and TensorFlow/PyTorch frameworks. Laboratory sessions focus on the complete machine learning workflow including data preprocessing, feature engineering, model training, evaluation metrics (accuracy, precision, recall, F1-score, ROC curves), and model interpretation techniques. Emphasis on real-world applications across domains such as finance, healthcare, marketing, and natural language processing. Course projects require students to develop end-to-end supervised learning solutions that address practical prediction problems on structured and unstructured data.

DS 444: Computing with Big Data

Prerequisites: DS 280

Comprehensive introduction to big data technologies and MLOps practices for deploying data science solutions at scale. This course focuses on both data engineering principles and the practical deployment of machine learning models. Topics include distributed computing frameworks, data lake architecture, cloud-based machine learning platforms, containerization with Docker, orchestration with Kubernetes, model deployment pipelines, model monitoring, and version control for data and models. Students will gain hands-on experience using cloud services to develop, deploy, and maintain scalable data science applications. Laboratory sessions provide practical experience with implementing end-to-end MLOps workflows, including model deployment, A/B testing, monitoring for drift, and automated retraining pipelines. Course projects require students to design and implement production-grade data science solutions that address real-world challenges while considering performance, scalability, reliability, and cost optimization. Emphasis is placed on industry best practices for taking data science models from development to production environments.
