Bachelor of Science in Robotics Engineering

Cognitive Robotics Track

The Robotics Engineering program at NJIT prepares students to develop the next generation of intelligent robotic systems. Our unique dual-track approach offers specialization in either Robotics Engineering (hardware and systems) or Cognitive Robotics (AI and perception), with a strong common foundation that ensures graduates understand the full robotics ecosystem.

Program Overview

The Cognitive Robotics track focuses on developing the "brain" of robotic systems. Students gain expertise in artificial intelligence, machine learning, computer vision, and human-robot interaction while maintaining a foundational understanding of robotics hardware and systems. This track prepares graduates for careers developing the intelligence that powers modern autonomous systems.

Robotics Engineering Track

Focus on hardware, mechanical systems, controls, and physical implementation of robotic systems.

Cognitive Robotics Track

Focus on artificial intelligence, perception, decision-making, and the cognitive aspects of robotic systems.

Program Distinctives

The Cognitive Robotics track at NJIT offers a unique interdisciplinary approach that combines the best of data science, computer science, artificial intelligence, and robotics engineering. By focusing on the intelligence that powers robotic systems while maintaining a solid foundation in robotics fundamentals, our program prepares graduates who can bridge the gap between AI theory and practical robotics applications.

Selected Course Descriptions

DS 113: Computing Foundations for Data Science (3 credits)

Course Description: Introduction to computational methods for data science using Python. Topics include implementing numerical integration techniques, solving application problems using definite integrals, computing infinite series approximations, working with exponential and logarithmic functions in data contexts, analyzing transcendental functions, visualizing polar coordinates, and computational approaches to conic sections. Laboratory sessions provide hands-on experience applying Python libraries such as NumPy, SciPy, Matplotlib, and SymPy to reinforce calculus concepts through programming.

Prerequisites: DS 100 or equivalent programming experience

DS 114: Computing Foundations for Data Science II (3 credits)

Course Description: Computational techniques in Python for data science with emphasis on linear algebra implementations and applications. Topics include matrix operations and manipulation using NumPy, implementing vector space operations, coding linear transformations, solving systems of linear equations programmatically, computing eigenvalues and eigenvectors, and applying these concepts to data science problems. Laboratory sessions reinforce mathematical concepts by visualizing abstract linear algebra operations, demonstrating the geometric interpretation of transformations, providing concrete computational examples of theoretical concepts, and showing how matrix decompositions connect to real-world data analysis.

Prerequisites: DS 113 and MATH 112 with a grade of C or better

CS 302: Human-Robot Interaction (3 credits)

Course Description: This course explores the theories, methods, and technologies for designing effective interactions between humans and robots. Topics include social robotics, natural interfaces, gesture recognition, affective computing, usability studies, and ethical considerations in human-robot collaboration. Students will design and implement interactive robot behaviors in various application domains.

Prerequisites: CS 114, RBTS 202

CS 401: AI for Robotics (3 credits)

Course Description: This specialized course bridges artificial intelligence techniques with practical robotics applications. Students will explore reinforcement learning, path planning algorithms, simultaneous localization and mapping (SLAM), and decision-making under uncertainty as applied to robotic systems. The course emphasizes implementation of intelligent algorithms for navigation, manipulation, and autonomous decision-making in both simulated and real-world robotic environments.

Prerequisites: CS 370, RBTS 201

DS 375: Supervised Machine Learning with Applications (3 credits)

Course Description: Comprehensive study of supervised learning algorithms and methodologies for predictive modeling. Topics include linear and logistic regression, decision trees, random forests, gradient boosting, support vector machines, neural networks, and ensemble methods. The course explores key concepts including bias-variance tradeoff, overfitting, regularization techniques, cross-validation, and hyperparameter tuning. Students will gain practical experience implementing these algorithms in Python using scikit-learn and TensorFlow/PyTorch frameworks. Laboratory sessions focus on the complete machine learning workflow including data preprocessing, feature engineering, model training, evaluation metrics, and model interpretation techniques.

Prerequisites: DS 240

Key Learning Outcomes

  • Design and implement AI algorithms for robotic perception, planning, and decision-making
  • Develop computer vision systems for robot perception and scene understanding
  • Create human-robot interaction interfaces that enable intuitive communication
  • Apply machine learning techniques to improve robot performance through experience
  • Integrate cognitive systems with robotic hardware for end-to-end intelligent systems
  • Understand ethical considerations in the development of autonomous systems
